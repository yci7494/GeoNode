## GeoNode 介绍

GeoNode是一家成立于2019年的专业代理服务提供商，主要为企业和个人开发者提供住宅代理、数据中心代理和移动代理服务。这家公司的核心优势在于其庞大的IP池资源——拥有超过200万个住宅IP地址，覆盖全球190多个国家和地区。
